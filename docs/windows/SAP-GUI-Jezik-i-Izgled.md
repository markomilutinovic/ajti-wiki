# SAP GUI — Podešavanje jezika i izgleda

Ovo uputstvo se primenjuje nakon instalacije SAP GUI 8.00. Podešava dva elementa:

1. **Jezik** — isključuje automatsko preuzimanje jezika SAP Logon-a na login ekranu
2. **Izgled (tema)** — postavlja SAP Signature Theme (standardni poslovni izgled)

---

## Korak 1 — Otvoriti Options

1. Pokrenuti **SAP Logon** sa desktopa.
2. Kliknuti levim dugmetom na ikonu menija (tri horizontalne linije) u gornjem levom uglu prozora.
3. Iz padajućeg menija izabrati **Options**.

## Korak 2 — Podešavanje jezika

1. U levom panelu proširiti **SAP Logon Options** → kliknuti na **General**.
2. U sekciji **SAP Logon Options** pronaći opciju:  
   **Use SAP Logon language as default on logon screen**
3. **Isčekirati** (ukloniti kvačicu) ovu opciju.
4. Kliknuti **OK**.
5. Ulogovati se normalno.

## Korak 3 — Podešavanje teme (izgleda)

1. Ponovo otvoriti **Options** (isti postupak kao u Koraku 1).
2. U levom panelu kliknuti na **Visual Design** → **Theme Settings**.
3. Pod **Theme Selection**, iz padajuće liste **Select Theme** izabrati:  
   **SAP Signature Theme**
4. Kliknuti **OK**.

## Korak 4 — Restartovati SAP Logon

Zatvoriti i ponovo pokrenuti SAP Logon kako bi se sve promene primenile.

> **Napomena:** Izmene teme se primenjuju tek pri sledećem otvaranju konekcije, ne odmah.
